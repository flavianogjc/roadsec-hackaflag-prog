# HACKAFLAG Fortaleza 2017
```
                   +++    HACKAFLAG - XOR     +++

 [+] Este desafio é bem simples. Dado um número N, gostaríamos de saber quantas
     duplas de números naturais (x, y) | x, y <= N, ao aplicar x XOR y, é possível obter o valor N.

     Exemplo: N = 5 

              0 XOR 5 -> 5
              1 XOR 4 -> 5
              4 XOR 1 -> 5
              5 XOR 0 -> 5

     A resposta correta é 4.

 [+] Como o desafio é muito simples, você terá apenas 3 segundos para
     responder cada desafio.
 
 [+] Para começar, digite start: start
      Iniciando...

 [+] Desafio 1 - N: 7
     A resposta é: 8
 [+] Correto!
 ```

## Execução
```
$ python2.7 hackaflag.py
```
