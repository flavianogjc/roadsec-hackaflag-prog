# HACKAFLAG Belo Horizonte 2017
```
                    +++    HACKAFLAG - TRIMOS   +++

 [+] Para desenvolvermos sistemas mais seguros, estamos testando a
    propriedade de alguns números para tentar utilizá-los em criptografias
    pós-modernas (pós-modernas, não pós-quânticas). Tentando dar um passo além,
    os números mais visados agora, para aplicações de segurança, são os números
    Trimos.

 [+] Se números primos são aqueles que possuem apenas dois divisores,
    números trimos são aqueles que possuem apenas três divisores.

 [+] Sua tarefa é nos ajudar respondendo se um número é trimo ou não.

 [+] Se o número for trimo, responda com 1, se não for trimo, 0.

 [+] O limite para cada resposta é de 1 segundos.

 [+] Para começar, digite start: start
      Iniciando...

 [+] Desafio 1 - N: 91
     A resposta é: 0
 [+] Correto!
 ```

## Execução
```
$ python2.7 hackaflag.py
```
