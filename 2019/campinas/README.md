# HACKAFLAG Campinas 2019
```
                           +++    TOCamisetas   +++

 [+] Todo evento, o ROADSEC vende camisetas personalizadas com temas relacionados a
     computação e cultura pop. Esse ano, Zbax ficou responsável pela realização das
     vendas na loja.

 [+] O problema é que Zbax tem TOC e sempre gosta de ver as camisetas extremamente
     organizadas. Então, ele organizou a estante de amostra em uma única linha onde
     cada coluna contém somente um tipo de camiseta e a mesma quantidade de camisetas
     cada uma.

 [+] Assim, sempre que termina uma palestra e uma rodada de camisetas são vendidas,
     algumas colunas ficam com mais ou menos camisetas que outras. Por causa do TOC,
     Zbax reorganiza a estante repondo ou retirando livros de forma que, ao terminar,
     sempre há a mesma quantidade de camisetas em todas as colunas.

 [+] Por querer tudo organizado o mais rápido possível, Zbax pediu a sua  ajuda
     para saber, dadas as quantidades de camisetas em cada uma das colunas na estante,
     qual o número mínimo de camisetas devem ser retiradas ou repostas para que todas
     as colunas fiquem com a mesma quantidade de camisetas.

 [+] Segue um exemplo:

     Estante desorganizada:                             [1 2 3 4 5 6]

     Estante organizada com menor número de movimentos: [4 4 4 4 4 4]

     Resposta:                                          9

 [+] O limite para cada resposta é de 2 segundos.

 [+] Para começar, digite start:  start
     Iniciando...

 [+] Estante 1/50: [1, 2, 3, 4, 5, 6]
     A resposta é: 9
 [+] Acertô, miseravi!
 ```

## Execução
```
$ python2.7 hack_a_flag.py
```
